# Canillo Dashboard (UI copy)

Simple, static dashboard UI that reproduces the layout and style from the provided screenshot. Uses Chart.js for working charts.

How to run
- Open `index.html` in your browser, or run a simple HTTP server from the project folder.
Open `index.html` in your browser, or run a simple HTTP server from the project folder.

Examples (from project root):
```powershell
cd "C:\Users\Student\Desktop\canillo_uno"
python -m http.server 8000

# then open http://localhost:8000 in your browser
```

This project now contains a simple To-Do app with localStorage persistence.

Files
- `index.html` — main To-Do page
- `css/styles.css` — styles for To-Do app
- `js/app.js` — To-Do logic (add, edit, complete, delete, filters, localStorage)
